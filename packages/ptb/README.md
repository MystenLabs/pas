# PTB Templates

This Move package defines helpers for describing programmable transaction block
(PTB) commands on-chain. PAS uses these templates so clients can automatically
build the approval calls required by permissioned assets.

## Docs

See the Sui documentation on
[setting template commands](https://docs.sui.io/onchain-finance/pas/integrating-pas#set-template-commands).

The TypeScript SDK is available as
[`@mysten/pas`](https://www.npmjs.com/package/@mysten/pas).
